# malloc_public

User documentation for **lilEasyFace**, a [lilToon](https://lilxyzw.github.io/lilToon/) shader add-on.

Built with [VitePress](https://vitepress.dev/) and published to GitHub Pages:
**<https://kanzaki1201.github.io/malloc_public/>**

This repository holds documentation and the issue tracker only — the shader source is not part of it.

## Report a bug or request a feature

Open an issue: <https://github.com/kanzaki1201/malloc_public/issues/new/choose>

## Local development

```bash
npm install
npm run docs:dev      # local preview at http://localhost:5173
npm run docs:build    # production build into docs/.vitepress/dist
```

Content lives in `docs/`. Push to `main` and GitHub Actions redeploys the site automatically.
